# Число прописью на C++

### Описание
Программа переводит числа в текст. Вводим цифру и получаем вывод прописью. Работает с числами от 0 до миллиона (включая отрицательные).

### Компиляторы
https://www.jdoodle.com/online-compiler-c++-ide

### Принцип работы

1.  **Массивы ("man", "fem", "teens" и другие):**
    Списки слов, т.е. "man" - для мужского рода (один, два), "fem" - для женского (одна, две — нужно для тысяч).

2.  **Окончание:**
    Чтобы понять, какое окончание ставить (миллион / миллиона / миллионов), используется функция "ending_idx". Она берет остаток от деления числа и по "ending" выбирает нужную форму.

3.  **Разбиение:**
    Большое число делится на миллионы, тысячи и сотни. Каждая часть переводится отдельно функцией "chunk_to_words", а потом всё склеивается в одну строку.
